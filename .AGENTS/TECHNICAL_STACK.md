# Stack Tecnológico y Dependencias

## Core
- **Kotlin**: 2.1.0
- **JVM Target**: 17
- **Android SDK**: Target 35 (Min 24)

## Librerías Clave
- **Jetpack Compose**: UI declarativa.
- **Room (2.6.1)**: Base de datos relacional local con soporte para Coroutines y Flow.
- **Firebase Firestore**: Base de datos NoSQL en tiempo real.
- **Firebase Auth**: Gestión de identidades (implementado de forma anónima para escalabilidad).
- **KSP (Kotlin Symbol Processing)**: Procesamiento de anotaciones rápido para Room.
- **Material Icons Extended**: Biblioteca completa de iconos para acciones de grupo y perfil.

## Componentes de Gradle
- **Google Services**: Plugin necesario para la comunicación con los servicios de Firebase.
- **MultiDex**: Habilitado para manejar el gran volumen de métodos de las librerías modernas.
- **AndroidX**: Migración completa a las librerías de soporte modernas.
